# Boulder Dash Recreation (Pygame)

Dette prosjektet er en rekreasjon av det klassiske spillet Boulder Dash, utviklet i Python med Pygame som en del av et skoleprosjekt i faget Programmering og modellering på videregående. Prosjektet ga meg tidlig erfaring med objektorientert programmering, logikk, datastrukturer og det å bygge et fullstendig, fungerende spill fra bunnen av.

## Om prosjektet

Spilleren navigerer seg gjennom nivåer, samler diamanter, unngår fallende steiner og prøver å nå utgangen. Jeg implementerte blant annet:

* Objektorientert struktur for spillobjekter

* Kollisjonsdeteksjon mellom spiller, steiner og terreng

* Spilloppdatering basert på et grid-system

* Enkel level-loader for å håndtere ulike brett

* Animert bevegelse og styring via Pygame med støtte for Commodore 64 joystick

* Noen enkle lydeffekter

* Enkelt leaderboard system

Prosessen lærte meg mye om strukturering av kode (eller kanskje mer hvor irreterende det er når det ikke er strukturert), debugging, og hvordan ulike deler av et spillsystem må samarbeide.
